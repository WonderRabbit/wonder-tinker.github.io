Wonder Tinker is a technical notebook for static Web publishing, AI-assisted
engineering workflows, and the verification artifacts that make small site
changes reviewable.

The site is intentionally source-oriented. Posts should describe what changed,
which tools or references informed the work, and how the result was checked.
